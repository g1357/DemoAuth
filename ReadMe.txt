Описание проектов
Web			- вёб-проекты
	DemoAuth	- 
	WebAPI		- Web API создан без управления пользователями. Добавил инфраструктуру Identity.
				Подключил использование БД SQL Server.
				Создал действие "login" для получения токена (жетона). Проверил через Postman.
	WebAppMVC	- Web Application MVC с управлением пользователями
	WebAppMVC2	- Web Application MVC с управлением пользователями
				и с добавлением всех страниц управления пользователями.
				База данных SQLite.
	WebAppRazor	- Web Application Razor с управлением пользователями

Mobile		- Мобильное приложение Xamarin.Form с использование Shell для платформ Android и iOS
				(UWP не поддерживается).

MobileWithUWP	- Мобильное приложение Xamarin.Form для платформ Android, iOS, UWP 
				для проверки работы вёб-сервиса WebAPI.
